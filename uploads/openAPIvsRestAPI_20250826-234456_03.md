## Example

You build this REST API:

    GET /users
    POST /users
    GET /users/{id}

You can define it in OpenAPI like this:

    paths:
      /users:
        get:
          summary: Get users
        post:
          summary: Create user
      /users/{id}:
        get:
          summary: Get user by ID
